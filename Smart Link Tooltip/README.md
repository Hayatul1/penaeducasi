## ✨ Fitur
- Tooltip interaktif dengan gambar dan teks
- Optimized performance dengan debouncing
- Responsive dan mobile-friendly
- Smart positioning otomatis

## 🚀 Cara Install

```html
<script src="smart-tooltip.js"></script>
```

```html

   data-title="Judul Tooltip">
```
